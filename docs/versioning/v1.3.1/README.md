# 版本 1.3.1

## 定位

1.3.1 是 1.3 的收尾与校准版本。

它不再扩大 Flow 的产品边界，而是把 1.3 已经确定的方向真正收干净，避免“架构上已经转向 FlowSpec，界面和实现里却还残留旧心智”。

## 目标

- 收掉 1.3 遗留的用户可见噪音和内部语义泄漏
- 完成一轮更系统的整体检测和回归
- 补齐 FlowSpec 用户层与实际实现之间仍然存在的断点
- 明确哪些问题属于 1.3.1 收尾，哪些问题进入 1.4

## 交付范围

本版本处理以下工作：

- 编辑器文案与交互收尾
- FlowSpec 主操作层的一致性校准
- 关键执行链路和导出链路的整体检查
- 回归测试与手工验收清单补齐

## 结果要求

- 用户默认路径不再暴露明显的内部语法和内部结构名词
- 关键主路径经过系统验证，而不只是局部功能通过
- FlowSpec 作为用户主操作层的定位，在文档、界面和代码里保持一致
