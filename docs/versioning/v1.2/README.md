# 版本 1.2

## 定位

1.2 版本聚焦 1.1 之后确认的实现修正和产品主路径收口。

本版本处理 bug、导出、测试、流式返回和 Agent 对话保存。

## 目标

- 修正 Plugin 默认输入语义相关问题
- 完成 Flow 和 Agent 的结构化导出
- 补上主路径测试
- 收口 Agent 流式返回体验
- 完成 Agent 对话保存

## 交付范围

本版本处理以下实现类工作：

- Flow 执行层和生成层的默认输入语义修正
- Flow 和 Agent 的结构化导出
- Agent 流式对话的前后端收口
- Agent 对话持久化
- 关键主路径测试补充

## 结果要求

- 新功能和修正不增加用户默认使用成本
- 导出的 Flow 和 Agent 结构清楚，适合人和模型理解
- 主路径有对应测试
- Agent 的流式状态和历史对话都可持续使用
