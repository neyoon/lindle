# Tweak

Tweak 是一个面向 AI 能力编排的工作台，提供可视化 Flow、对话式 Agent、Skill 和 Plugin 机制。

## 核心能力

- 可视化编排 Flow
- 用 Agent 调度 Skill
- 让 Flow 作为 Agent 能力被调用
- 支持 OpenAI 兼容模型接口
- 支持导出独立运行的工作流项目

## 快速开始

前置要求：

- Python 3.11+
- Node.js 20+
- uv

启动：

- 使用项目根目录中的启动脚本进入开发模式
- 需要热更新时使用热更新模式
- 需要接入真实认证服务时使用真实认证模式

启动后访问本地前端入口。
